Simple Form Validation

Exercise 1

Create a form with a name and email field. When the form is submitted, write JavaScript code
to check if both fields are filled in. If not, display an error message below to the empty field(s) in red colors. 

   
  const Fname = document.getElementById('Fname');
  const email = document.getElementById('email');
  const emailWarning = document.getElementById('emailWarning')
  const nameWarning = document.getElementById('nameWarning')
  
  function check() {
      if (!Fname.value) {
          nameWarning.style.display = "block";
      } else {
          nameWarning.style.display = "none";
      }
  
      if (!email.value) {
          emailWarning.style.display = "block";
      } else {
          emailWarning.style.display = "none";
      }
  }
